# Contexto do Profissional — Front-end com Especialização em UI e UX

## Visão Geral

Este projeto deve ser conduzido por um **desenvolvedor front-end** com **especializações práticas em UI (User Interface) e UX (User Experience)**. Não se trata de três cargos separados, mas de um único perfil técnico que compreende **como código, interface e experiência do usuário se conectam**.

O objetivo desse profissional é garantir que o produto final:

* Seja **visualmente consistente**
* Tenha **boa experiência de uso em mobile e desktop**
* Seja **performático, acessível e sustentável tecnicamente**

---

## Papel Principal: Front-end Developer

O front-end é o responsável direto por **transformar design e intenção de experiência em código funcional**.

Neste projeto, isso significa:

* Escrever **HTML semântico**
* Estruturar **CSS moderno e escalável**
* Implementar **JavaScript focado em experiência, performance e interação**

Esse profissional **não apenas implementa layouts**, mas entende **como o usuário interage com o site** e como o navegador processa cada decisão técnica.

---

## Linguagem Principal: JavaScript

### Por que JavaScript?

JavaScript é a linguagem responsável por:

* Interações (scroll, clique, swipe)
* Animações e transições
* Estados visuais (ativo, oculto, expandido)
* Comportamento dinâmico da interface

No Magnolia Tape, o JavaScript é usado para:

* Controle de header dinâmico
* Animações baseadas em scroll
* Expansão de membros do time
* Carousel com autoplay, swipe e navegação

### Responsabilidade técnica

O front-end precisa entender que **JavaScript impacta diretamente**:

* Performance (Lighthouse)
* Experiência em mobile
* Consumo de bateria
* Acessibilidade

Portanto, decisões em JS **não são apenas técnicas**, são decisões de experiência.

---

## Especialização em UI (User Interface)

UI é a especialização que garante que a interface:

* Seja clara
* Seja consistente
* Seja legível
* Respeite hierarquia visual

Neste projeto, a especialização em UI envolve:

* Tipografia (uso correto de fontes display e body)
* Espaçamentos e respiros visuais
* Cores e contraste
* Estados visuais (hover, active, focus)

O front-end com domínio em UI entende que **cada classe CSS comunica algo ao usuário**, mesmo quando ele não percebe conscientemente.

---

## Especialização em UX (User Experience)

UX é a especialização que garante que o usuário:

* Entenda o site rapidamente
* Saiba o que fazer
* Não sinta fricção ao navegar

No Magnolia Tape, UX envolve:

* Ordem lógica das seções
* Momento correto de exibição do CTA
* Comportamento previsível de animações
* Fluidez no scroll

Aqui, o front-end precisa pensar:

> “Essa animação ajuda ou atrapalha a leitura?”

UX não é estética — é **comportamento e percepção**.

---

## Conexão entre Front-end, UI e UX

Neste projeto, **não faz sentido separar completamente essas áreas**.

O profissional ideal entende que:

* UI sem performance vira ruído
* UX sem código vira teoria
* Código sem UX vira site difícil de usar

Por isso, o perfil esperado é:

**Front-end developer com visão de produto**, capaz de:

* Tomar decisões técnicas conscientes
* Ajustar código pensando em pessoas reais
* Usar JavaScript, CSS e HTML como ferramentas de experiência

---

## Objetivo Final do Profissional

Garantir que o Magnolia Tape seja:

* Visualmente forte
* Tecnicamente sólido
* Experiencialmente fluido

Não apenas um site bonito, mas um **produto front-end bem construído**.
